# T-SQL Querying Lab

This repository contains practical T-SQL queries and database programming concepts covered during the T-SQL Querying Laboratory course. The lab exercises are organized unit-wise to help understand SQL fundamentals, advanced querying techniques, database objects, and error handling mechanisms.

## 📚 Course Contents

### Unit 1: Introduction to SQL

* SQL Basics
* DDL Commands (CREATE, ALTER, DROP)
* DML Commands (INSERT, UPDATE, DELETE)
* DQL Commands (SELECT)
* Filtering and Sorting Data
* Aggregate Functions
* Grouping Data

### Unit 2: SQL Constraints, Joins & Views

* Primary Key
* Foreign Key
* Unique Constraint
* Check Constraint
* Default Constraint
* Types of Joins

  * Inner Join
  * Left Join
  * Right Join
  * Full Join
  * Cross Join
* Views and their Applications

### Unit 3: Window Functions and CTE

* ROW_NUMBER()
* RANK()
* DENSE_RANK()
* NTILE()
* LEAD() and LAG()
* Common Table Expressions (CTE)
* Recursive CTE

### Unit 4: Stored Procedures & User Defined Functions

* Creating Stored Procedures
* Executing Stored Procedures
* Parameterized Procedures
* Scalar Functions
* Table-Valued Functions
* Function Usage Examples

### Unit 5: Cursor, Trigger & Exception Handling

* Cursor Operations
* Types of Cursors
* Triggers

  * AFTER Trigger
  * INSTEAD OF Trigger
* TRY...CATCH Blocks
* Error and Exception Handling
* Transaction Management

## 🛠️ Tools Used

* Microsoft SQL Server
* SQL Server Management Studio (SSMS)
* T-SQL

## 🎯 Objective

The objective of this repository is to provide a collection of T-SQL laboratory exercises and examples for learning and practicing SQL Server database programming concepts.
